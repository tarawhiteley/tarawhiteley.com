# Dependencies
You'll need the following installed on your computer:

* Node
* Bower
* Gulp
* Yarn


# QuickStart

1. Clone this repository.
2. Navigate to your project folder within the Terminal and run `yarn install`.
3. Run `bower install`.
4. Run `gulp`.
